PowerBand HTML Template Documentation

ThemeForest-ready HTML template package.

Template Overview

PowerBand is a responsive HTML template for fitness and gear brands. It ships with clean markup, Tailwind Play CDN (bundled locally), local fonts, an SVG icon sprite, and a modular asset structure.

Included pages: index.html, about.html.

Requirements

  • Modern browser (Chrome, Firefox, Edge, Safari).
  • Optional local server for best results (recommended).

Installation / How to Run

  • Open Template/html/index.html directly in your browser.
  • Or use a local server for best results (recommended):
cd Template/html
# Python
python -m http.server 8000
# Then open http://localhost:8000/index.html

Editing Guide

Change Logo

Replace the logo icon in Template/html/index.html by swapping the SVG reference in the navbar and footer.

Change Colors

Update CSS variables in Template/assets/css/style.css and Tailwind utility classes in the HTML.

Change Fonts

Replace the font files in Template/assets/fonts and update @font-face in Template/assets/css/style.css.

Update Navigation

Edit anchor links in the navbar to match section IDs.

Add New Sections/Pages

Copy existing sections in Template/html/index.html and adjust IDs. For new pages, copy index.html or about.html, update content, and keep the shared header/footer.

Dependencies & Plugins

FAQ & Troubleshooting

  • Icons not showing? Ensure Template/assets/img/icons.svg is present and relative paths are intact.
  • Fonts not loading? Confirm the font files in Template/assets/fonts match the @font-face URLs.
  • Tailwind classes not rendering? Verify Template/assets/vendor/tailwindcss/tailwindcss.js is included in the head.
  • Contact form? This template includes a newsletter form only. Add your own backend integration if needed.
